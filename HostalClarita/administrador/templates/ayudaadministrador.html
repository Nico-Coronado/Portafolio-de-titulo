SOPORTE DEL HOSTAL CLARITA

Este manual de ayuda sirve para cada una de las funciones que usted puede ver en la barra de navegación superior. Haga click en el manual que le interese y se desplegará la explicación.

¿Con qué necesitas ayuda?

Manual sistema de Habitaciones

Este sistema de habitaciónes tiene disponible una lista con el cual podra agregar una habitación por su numero de habitación, una breve descripcion, la disponibilidad de la misma y por ultimo el precio. Una vez agregada la habitación esta se vera reflejada en una tabla inferior donde se podran ver todos los atributos agregados, ademas de poder actualizar la información y eliminar con los botones de accion que tiene a la derecha de esta misma tabla.

Manual sistema de Huéspedes Actuales

Sistema que le muestra todos los huéspedes que se encuentran en el hostal.

Manual Sistema Libro de Huéspedes

Sistema que muestra a todos los huéspedes qué ha tenido, tiene y tendrá el hostal.

Manual Sistema de Facturas

Este sistema permite la anulación de las facturas. Esta zona es dedicada por lo que se le pregunta antes de querer borrarla.

Manual Sistema de Comedores

Este sistema permite crear, buscar, actualizar y eliminar comedores de servicio, en caso de que quiera buscar actualizar o eliminar debera usar una id para buscar el comedor, la cual sera otorgada una vez se cree un comedor.

Manual Sistema de Platos

Este sistema tiene disponible una lista con la cual podrá agregar el nombre del plato, el dia en el que se servirá el mismo, la hora, el precio correspondiente del plato y por último una id con la cual reconocerlo. Una vez agregado el plato en la parte inferior se desplegará una tabla con los datos ingresados, además de poder eliminar y actualizar los datos de los platos con unos botones de accion que tiene a la derecha de esta misma tabla.

Manual Sistema de Pedidos

Este sistema muestra todos los pedidos que se han realizado en el hostal, se pueden revisar y eliminar.

Manual Sistema de Productos

Este sistema muestra todos los productos que el hostal puede pedir a todos sus proveedores.

Manual Sistema de Familia de Producto

En este sistema se puede crear, buscar, actualizar y eliminar familias de productos, en caso de que quiera buscar actualizar o eliminar debera usar una id para buscar una familia, la cual sera otorgada una vez se haya creado una familia de productos.

Manual Sistema de Monedas

En este sistema se puede crear, buscar, actualizar y eliminar monedas de servicio, en caso de que quiera buscar actualizar o eliminar debera usar una id para buscar la moneda agregada, la cual sera otorgada una vez se haya creado una moneda.

Manual Sistema de Estadisticas

Este sistema entrega tres informes importantes para la gestión de nuestro hostal: nos da las ganancias que hemos tenido mensual y anualmente, también cálcula los gastos que hemos tenido mensual y anualmente. Además muestra las visitas que ha tenido la página web del hostal.

Manual Sistema de Registro de Usuarios

En este sistema usted puede registrar a nuestros empleados o proveedores. También se le da la opción de crear clientes.

Manual de sistema de Módulos.

Aquí puede acceder a la vista y funcionamiento de los siguientes tipos de usuarios: Cliente, Proveedor y Empleado.

Manual de sistema de Hostal.

Aquí puede actualizar el hostal de manera manual presionando sólo un botón, ese botón asignará y liberará habitaciones acorde los días reservados por los huéspedes.
